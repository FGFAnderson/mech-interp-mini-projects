# Einops Drills

Practice drills for getting fluent with `einops` (`rearrange`/`reduce`)

See [Einops.ipynb](Einops.ipynb):

- **Part 1** — warm-up drills (NCHW↔NHWC, flattening, splitting/merging attention heads, global average pooling)
- **Part 2** — ViT-style patch embedding (image → sequence of flattened patches)
- **Part 3** — debug-the-pattern: spot the bug in broken `rearrange` calls

## Setup

```bash
uv sync
```
